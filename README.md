# PHP CRUD - Update Operation

## Features
- Loads existing user data from the database
- Displays a form with editable fields
- Updates the user using a prepared SQL statement
- Shows success/failure messages

## How to Run
1. Import `users.sql` (optional).
2. Update database settings in `db.php`.
3. Run using XAMPP or WAMP.
4. Open: http://localhost/crud-update/edit_user.php?id=1
